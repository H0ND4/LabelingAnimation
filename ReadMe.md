# 画像処理 ラベリング処理
ラベリングとは？
- 領域ごとに画素を分ける処理を指す
- 今回は色分け表示して同じ領域を同じ色とする
- 白画素を前景領域とする
## 処理過程をアニメーション化して，過程を理解する

ラスタ走査を行い，随時ラベリング処理を行う．
処理過程画像がanimフォルダに保存される．

makeGIFAnimation.pyを使用して
画像からGIFアニメーションを生成する
